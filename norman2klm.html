Model Comparison Matrix

Comparison of Model Scope (Dix et al., Ch 12)
Model Category What it Explains What it Predicts
Norman Cognitive / Descriptive Understanding, errors, and feedback gulfs. ❌ No quantitative timing.
GOMS Predictive / Analytic Task structure and selection rules. ✔ Relative efficiency.
KLM Predictive / Quantitative Low-level physical/motor actions. ✔ Exact execution time.

KLM Operators: The Seven Primitives

The Keystroke-Level Model (KLM) is a low-level instance of GOMS. It assumes expert, error-free performance.

Seven Operators of KLM
Code Operator Description Example
KKeystrokingPressing a keyTyping 'A'
BButton PressMouse clickLeft-click
PPointingMoving the cursorMoving to 'Save'
HHomingSwitching deviceHands to Mouse
DDrawingManual drawingDrawing a line
MMental PrepCognitive pauseThinking before click

Sequence for deleting and retyping one character using a mouse:

P B[LEFT]

Key Takeaways

KLM only models execution, not high-level planning. It is best used for comparing the speed of two different interface designs for the same task.
